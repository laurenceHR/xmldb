Xmldb

PHP Library for use XML as DataBase Storage

Download .zip Download .tar.gz View on GitHub

Hola, este proyecto es una librería para facilitar y simplificar el almacenamiento de datos, sin necesidad de un motor de base de datos (como MySQL), pues a veces son pocos y simples los datos que queremos almacenar (login de usuarios, números telefónicos).

Pero a la vez el manejo de este almacenamiento permite hacer operaciones SQL comunes (SELECT,UPDATE,DELETE) para que sea de fácil uso.

No domino mucho Git así que espero poder subir el código, sino lo publicaré en mi web.
